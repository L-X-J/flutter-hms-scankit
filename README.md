# flutter_scankit

A scan code Flutter plugin, which is a Flutter package for [HUAWEI Scan Kit](https://developer.huawei.com/consumer/en/doc/development/HMSCore-Guides-V5/service-introduction-0000001050041994-V5) SDK.

- [x]  Android
- [ ]  iOS
